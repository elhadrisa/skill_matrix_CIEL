Tutoriel complet CIEL Matrix

Application en ligne : https://cielmatrix.fr

CIEL Matrix est une application de pilotage pédagogique dédiée au Bac Pro CIEL. Elle permet de gérer les classes, les élèves, les compétences, les séances, les PFMP, les remédiations, les exports et les synthèses annuelles sur les 3 années du parcours.

2nde 1ère Terminale Compétences PFMP Exports

1. Se connecter

Ouvre le site à l’adresse https://cielmatrix.fr. La page de connexion permet de saisir un identifiant et un mot de passe. Selon le rôle attribué, l’utilisateur accède à des droits différents :

  • Administrateur : gestion des comptes, structure, classes, élèves, exports complets.
  • Prof principal : gestion pédagogique avancée.
  • Professeur : saisie des compétences, évaluations, PFMP.
  • Lecture seule : consultation sans modification.

2. Comprendre la navigation

Topbar

La barre du haut donne accès à tous les modules : Classes, Dashboard, Evaluations, PFMP, Bulletin, Référentiel, Remédiations et Comptes pour l’administrateur.

Menus déroulants

Les menus Dashboard, Evaluations, PFMP et Bulletin sont classés par classe. Cela permet d’ouvrir directement la bonne vue sans charger une page trop lourde.

3. Gérer les classes et les élèves

Le module Classes sert à créer les classes et à ajouter les élèves.

  1. Créer une classe avec son nom, son année et sa note de contexte.
  2. Ajouter les élèves un par un.
  3. Importer des élèves en masse par fichier CSV.
  4. Modifier ou supprimer une classe ou un élève.

Exemple de classes : 2CIEL, 1CIEL, TCIEL

4. Suivre les compétences

Les compétences du référentiel sont suivies élève par élève avec les niveaux suivants :

  • Absent
  • Non évalué
  • Non acquis
  • En cours d’acquisition
  • Partiellement acquis
  • Acquis

Les domaines métiers utilisés dans l’application sont :

Réseau Informatique Cybersécurité Electronique

5. Créer et évaluer une séance

Création de séance

Dans Evaluations, choisis la classe puis Création de séance. Tu peux définir :

  • type de séance : TP ou TD
  • titre
  • date de début et date de fin
  • plusieurs compétences associées
  • plusieurs indicateurs
  • banque d’indicateurs par domaine

Evaluation de séance

Dans Evaluation séance, sélectionne la séance créée puis évalue chaque élève indicateur par indicateur. Les résultats remontent ensuite dans les bilans et les synthèses.

6. Utiliser le dashboard

Le module Dashboard est organisé par classe avec trois vues principales :

  • Pilotage compétences : progression, répartition, synthèse.
  • Pilotage PFMP : conventions, visites, dossiers complets.
  • Calendrier pédagogique : affichage des séances et des échéances.

Le calendrier prend en compte les dates de début et de fin des séances. Un clic sur une date permet de préparer une création de séance.

7. Gérer les PFMP

Infos PFMP

Le module PFMP permet de suivre, pour chaque élève et chaque période :

  • entreprise
  • tuteur
  • conventions
  • date de visite
  • rapport, livret, présence
  • compétences observées

Livret d’évaluation

Le livret PFMP est séparé des informations administratives. Il sert à l’évaluation PFMP selon ton fonctionnement réel d’établissement.

8. Utiliser les remédiations

Le module Remédiations distingue :

  • Remédiations PFMP
  • Remédiations compétences

Les alertes peuvent être filtrées, exportées en PDF et utilisées pour créer directement une séance de remédiation.

9. Référentiel, cartographie et bibliothèque

Couverture du référentiel

Cette vue montre si les compétences sont suffisamment travaillées selon la classe et le domaine.

Cartographie

La cartographie relie les compétences aux séances et aux PFMP. Une heatmap permet de visualiser la couverture.

Bibliothèque de séances

La bibliothèque stocke des modèles de séances réutilisables, classés par domaine, niveau et compétences.

Duplication

Un modèle peut être dupliqué vers une classe, avec une période et des dates, pour accélérer la création des TP/TD.

10. Exports et bilans

L’application permet d’exporter :

  • les bilans de compétences par classe
  • les PFMP par classe
  • les bilans élève par année
  • les synthèses sur les 3 ans
  • les PDF de séance, remédiation, PFMP et bulletin

Les exports disponibles suivent maintenant les 3 niveaux :

2nde 1ère Terminale Synthèse 3 ans

11. Gestion des comptes

Le module Comptes est réservé à l’administrateur. Il permet :

  • d’ajouter des comptes
  • de modifier les identifiants et mots de passe
  • de choisir le rôle
  • de supprimer des comptes
  • d’exporter ou restaurer une sauvegarde

12. Bonnes pratiques d’utilisation

  • Créer les classes avant d’importer les élèves.
  • Saisir les séances régulièrement pour éviter les bilans incomplets.
  • Mettre à jour les PFMP au fil de l’eau.
  • Utiliser les remédiations pour préparer les prochaines séances.
  • Faire des exports réguliers pour conserver un historique externe.
