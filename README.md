# Application Bac Pro CIEL

Application web autonome pour suivre les compétences d'élèves en Bac Pro CIEL.

## Ouvrir l'application

Il suffit d'ouvrir `index.html` dans un navigateur.

## Fonctionnalités

- tableau de bord global
- sélection d'élève
- filtrage par domaine et niveau
- mise à jour des compétences
- sauvegarde locale via `localStorage`

## Fichiers

- `index.html`
- `styles.css`
- `app.js`
